Trocker has been updated! Here are the changes:
v2.0.0:
  • Special Gmail treatment. In Gmail, an icon is shown on top of each email that is found to have a tracker in it.
  • Context aware in Gmail; won't expose trackers in print view or compose windows.
  • Heuristic tracker detection. Now any tiny image that looks like a tracker is blocked in Gmail.
  • New advanced options for those who want to swim deeper. Now only enables logging and shows the log of heuristic detections.
  • Block even more trackers (now MailChimp, SendGrid and iContact in addition to MailTrack, Mandrill, Yesware and SideKick)
v1.2.2:
  • Block even more trackers (now MailTrack and Mandrill in addition to Yesware and SideKick)
v1.2.1:
  • Stop tracking extensions from reloading the tracking url in background (not possible if extensions use their background page to reroute their tracking)
v1.2.0:
  • Blocking more Trackers! (now Sidekick in addition to Yesware)
  • Trocker now also bypasses tracked links (links that inform the sender when you click on them) as well as tracked emails (emails that inform the sender when you open them). If possible, Trocker bypasses the intermediate link that is used to track link clicks and redirects you directly to the original link.
  • All new click tracker bypasser page
  • Exposing link trackers by adding a "T" to the mouse pointer
  • Support Outlook.com's image proxy
  • Improvement in exposing open trackers (specially in compose window).
  • Improved options page.
v1.1.1:

  • Don't open changelog on small updates
  • Bugfix for new installations.

v1.1.0:

  • Open changelog in a new tab on update

v1.0.6:

  • Extension's name was changed to Trocker in order to avoid similarities with trademarked names.
  • Updated look and feel of the app.
  • Icon only shows the number of trackers found on the current tab.
  • New options page (for chrome v42 and later)

v1.0.5:

  • Exposing Tracker Links is disabled by default. You can enable it from the options page if you like!
  • Updated Icon for disabled extension.

v1.0.4:

  • Improvements in exposing tracker links.

v1.0.3:

  • The extension can now make tracking images visible. You can disable this feature in the options page.
